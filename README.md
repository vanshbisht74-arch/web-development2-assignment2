# Weather App (Async Project)

## About

This is a simple weather app made using HTML, CSS and JavaScript.
It shows weather of a city using an API.

This project is made to understand:

* async and await
* fetch API
* event loop (microtask and macrotask)
* localStorage

---

## Features

* Search weather by city name
* Shows temperature, weather, humidity and wind
* Saves search history
* Click history to search again
* Console shows async working

---

## How to Run

1. Download all files
2. Keep files in same folder:

   * index.html
   * style.css
   * script.js
3. Open index.html in browser

---

## How it Works

* User enters city name
* App calls weather API using fetch
* Data is shown on screen
* City is saved in localStorage
* Logs show async steps
